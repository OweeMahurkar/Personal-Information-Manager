# Personal Information Manager

This is a complete Python program built to store, process, and display personal data. It demonstrates the use of variables, basic input/output, while-loop input validation, and string formatting using f-strings.

## Features
- Stores static personal details (Name, Age, City, Hobby) using correct data types.
- Captures user input for favorite things with validation loops to prevent empty submissions.
- Dynamically calculates the stored age in months.
- Automatically cleans and formats text output using string methods like `.title()` and `.lower()`.

## Project Structure
- personal_info.py : The main Python script.
- README.md : Project documentation and setup instructions.
- test_inputs.txt : A log documenting different test cases and edge cases.
- .gitignore : A file instructing GitHub to ignore temporary or hidden system files.

## How to Run
1. Ensure you have Python installed on your system.
2. Open your terminal or command prompt.
3. Navigate to the project directory.
4. Run the program using the command: python personal_info.py
